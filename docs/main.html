Project Overview

This project involves a hardware-in-the-loop setup to test and validate a LM2596 DC-DC converter under different load conditions using LabVIEW.

Hardware Components

  • DC-DC Converter (LM2596)
  • NI USB-6001 DAQ Module
  • Load Resistors (3.3Ω, 4.7Ω, 6.8Ω, 10Ω)
  • Current Limiter (LM317 Regulator)

Installation

  1. Clone the repository: git clone https://github.com/yourusername/hil-dut-tester.git
  2. Install LabVIEW and necessary DAQ drivers.
  3. Connect the DUT and DAQ module as per the circuit diagram.
  4. Run the LabVIEW project.
